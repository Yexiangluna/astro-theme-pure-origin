# 快速开始

欢迎使用 Astro Theme Pure 主题！本指南将帮助你快速设置和运行你的博客。

## 前置要求

- Node.js 18+ 或 Bun
- 包管理器 (npm, pnpm, yarn 或 bun)

## 安装

```bash
# 使用 bun
bun create astro@latest --template cworld1/astro-theme-pure

# 或克隆仓库
git clone https://github.com/cworld1/astro-theme-pure.git
cd astro-theme-pure
bun install
```

## 启动开发服务器

```bash
bun dev
```

访问 `http://localhost:4321` 查看你的网站。
